Welcome to Fungitrack!

Mushroom collection can be fun, delicious and profitable, and to that end, we are building a database for finding the best mushroom picking locations, and sharing them with your friends. It will help you learn what mushrooms are in season and where to find them.

For the more technically inclined, here is a quick overview of our current technology stack

Python

Programming language with good documentation, multiple paradigm support, a short learning curve, and excellent libraries.

Flask

Minimalist python web library which support quick iteration and minimal code to get up and running.

Jinja2 (included in flask)

Common templating language for python that comes with Flask, which allows you to have all the convenience of declarative HTML construction.

WTForms

Form validation system, which makes it easy to validate web form data and verify that all the information is correct, and then fix anything you did wrong.

Bootstrap

Frontend web development toolkit; makes it easy to make web pages that are not ugly!

Pip

Python package management system. Installing it can be a bit of a hassle, but once you are done, a simple "pip install -r requirements.txt" ensures you have all the libraries needed to run code.

MySQL

... not my first choice, but easy to deal with since he is providing us with the DB I guess? For now, SQLite will suffice.

MySQL-connector-python

A pure python implementation of a MySQL connector, and therefore easy to import

Git + Github

A preeminent platform for distributed version control. Make an account and I will give you push privilages.
